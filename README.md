These are drafts for a lecture on an intuitive introduction to probability for 
Computational Finance using R. The basic idea and spirit of this course is to
develop an understanding probability concepts through building them piece by piece
on a computer and play with them in this way. The tool for building and playing will
be R and RStudio. We will build R skills in parallel with the theoretical 
probability concepts.

The outline of our probability syllabus follows the organisation of Karl 
Schmedder's excellent coursera course "An Intuitive Introduction to Probability: 
Decision-Making in an Uncertain World". We will attempt to choose the examples such
that they have a focus on Finance and computational Finance applications.
The major sections are:

1. Probability Theory
2. Conditional Probability
3. Random Variables
4. The Normal Distribution

R concepts that should be developed in parallel with these probability topic should
include:

- Interface: R and RStudio
- objects
- functions
- scripts
- packages 
- getting help
- atomic vectors
- attributes
- matrices
- arrays
- class
- coercion
- lists
- data.frames
- loading and saving data
- selecting values
- modifying values (subsetting)
- missing values
- environments (scoping, assignments, evaluation and closure)
- programs
- S3
- loops; speed
